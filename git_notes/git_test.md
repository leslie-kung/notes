### 测试git 的版本回退问题

**version1：111111111**


### 本地分支回退
```
# 查看历史版本信息
git log
# 单行显示
git log --pretty=oneline
```

```
# 方法一
# 回退上个版本(^ 表示回到上一个版本， ^^表示回到上上一个版本，~10 表示回到前10个版本)
git reset --hard HEAD^

# 方法二
 git reset --hard  版本号
```

#### 假如从第三个版本回退到了第二个版本，又想回到第三个版本
```
# 获取之前的版本号
git reflog
# 回退
git reset --hard 版本号
```

### 远程仓库版本回退
```
# 本地版本回退后 -f强制远程仓库版本回退
git push -f
```
